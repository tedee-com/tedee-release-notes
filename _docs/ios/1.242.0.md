## Bridge Wi-Fi module firmware update

The Bridge contains a dedicated Wi-Fi module responsible for its internet connectivity. Starting with this version, the Wi-Fi module firmware can be updated in two ways:

- **During the bridge adding process** — if an update is available, it will be applied as part of setting up your Bridge.
- **From Bridge settings** — the current Wi-Fi module firmware version is visible in the Bridge firmware settings page, and an update can be triggered from there whenever a new version is available.

This ensures your Bridge stays fully up to date, including its wireless communication component, without any need for manual intervention outside of the app.

## Bluetooth-only mode now activates only during major Remote Control Service outages

Previously, any incident affecting the Remote Control Service component would automatically switch the app to Bluetooth-only mode, regardless of its severity. This could unnecessarily restrict remote access even during minor or partial disruptions.

With this update, automatic Bluetooth-only mode is activated only when the Remote Control Service reaches a **Major Outage** status. Depending on the severity of an incident, the app will now behave as follows:

- **Major outage**: Bluetooth-only mode is enabled automatically. The incident status bar will display relevant information about the outage, and a red dot indicator will appear pointing to the Bluetooth-only mode settings.
- **Minor or partial incident**: Bluetooth-only mode is **not** enabled. The incident status bar will still be shown, but with messaging appropriate to the lower severity — and remote access remains available.

This change ensures that remote access is preserved wherever possible, and Bluetooth-only mode is reserved for situations where remote operations are genuinely unavailable.

## Improved app layout for users with large font sizes

For users who rely on larger font sizes configured in **iOS Settings → Accessibility**, various parts of the app have been improved to ensure all information is displayed correctly and completely. Previously, some UI elements, labels, and content across different screens could be cut off, overlapping, or otherwise difficult to read when a larger system font size was selected.

This update addresses layout issues across multiple screens, making the Tedee app more accessible and readable for users who depend on increased text size for better visibility.
