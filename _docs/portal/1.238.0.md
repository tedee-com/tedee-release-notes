## Adjustments

* **Guesty integration now uses API key**: The method of connecting to your Guesty account has changed. Instead of Client ID and Client Secret, you now need to provide an API key to establish the integration. You can find your API key in the Guesty platform settings. More details on how to configure the integration are available in the [knowledge base](https://tedee.com/knowledge-base/integration-of-tedee-with-channel-management-system-guesty/).
