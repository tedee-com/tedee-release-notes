## Features

* **New event type**: Wrong PIN entry limit reached events are now tracked and displayed in the activity logs.
* **Keyboard shortcuts for Groups management**: Streamline group operations with three new keyboard shortcuts available exclusively on the Groups page:
  - **(N)** - Quickly add a new group from the group list page
  - **(Enter)** - Confirm and save the new group
  - **(Esc)** - Cancel group creation and close the sidebar

  Hover over action buttons to see helpful tooltips displaying the corresponding keyboard shortcuts.

  ![Shortcut tooltip example](/tedee-release-notes/docs/portal/assets/1.233.0_shortcut-tooltip-example.png)

## Improvements

* **Enhanced CSV export**: Organization users export now utilizes the backend endpoint for improved performance.

## Fixes

* Fixed duplicate export functionality and resolved issue where user emails were not selectable on the `Device > Users` page.
* Corrected users list sidebar height and restored ability to select user email addresses.
