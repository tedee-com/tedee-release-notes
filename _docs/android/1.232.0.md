## Disable unlocking from devices list for preview users
Preview users can now disable the ability to unlock devices directly from the devices list.

## External power not available for Keypad V1
Removed external power setting for Keypad V1 devices.

## Fixed unauthorized auto-unlock
Resolved an issue where auto-unlock could be triggered without proper authorization.
