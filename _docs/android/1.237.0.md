## Lock Go: Matter support
Lock Go now supports Matter. This enables seamless integration with Matter-compatible smart home ecosystems, allowing easier setup, cross-platform control, and more reliable connectivity.

## Multiple Wi-Fi networks for Auto-unlock
You can now add several Wi-Fi networks (e.g., 2G, 5G) for auto-unlock by separating names with commas in the setup. The description now explains this option.

![screenshot](/tedee-release-notes/docs/android/assets/1.237.0_wifi_items.jpg)

## Improved privacy policy popup
The privacy policy popup automatically expands and uses larger fonts when device font scaling is enabled, ensuring all content is visible and accessible.
