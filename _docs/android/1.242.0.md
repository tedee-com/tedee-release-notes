## Bridge Wi-Fi module firmware update

The Bridge contains a dedicated Wi-Fi module responsible for its internet connectivity. Starting with this version, its firmware can be updated in two ways: automatically during the Bridge adding process if an update is available, or manually from the Bridge firmware settings page. This ensures your Bridge stays fully up to date, including its wireless communication component.

## Bluetooth-only mode now activates only during major Remote Control Service outages

Previously, any incident affecting the Remote Control Service would automatically switch the app to Bluetooth-only mode, regardless of severity. With this update, Bluetooth-only mode is activated automatically only when a **Major Outage** is detected — preserving remote access during minor or partial incidents while still displaying the incident status bar with appropriate messaging.

## Quick calibration restore for Tedee Pro 2

When a Tedee Pro 2 lock loses its calibration but previous calibration data is still available, the app now offers a quick restore option. Instead of always requiring a full calibration process, you can press the lock button to automatically recover the previous calibration settings. If the quick restore fails, the app guides you through a full calibration. This experience is now aligned with Tedee GO 2.

## Improved app layout for users with large font sizes

Various screens have been updated to correctly display all content when a large system font size is configured in **Android Settings → Display**. Previously, some labels and UI elements could appear cut off or overlapping at increased text sizes. This update improves readability and accessibility across the app.

## Improved activation code readability

The activation code entry screen now uses improved text contrast, making the displayed code easier to read. Previously, the font color was too light against the background, making it difficult to distinguish characters under normal viewing conditions.

## Bug fixes

- Fixed an issue where home screen widgets could intermittently show a logged-out state even though the user was still signed in.
- Fixed an issue where Admin users in Teams could not add new fingerprints — the fingerprint screen would remain in a loading state indefinitely.
