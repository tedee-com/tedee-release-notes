## New lock and unlock button design
The refreshed lock and unlock button design, previously available to preview testers, is now rolled out to all users.

## Added a toggle to enable or disable haptic feedback in the main application settings.
You can now customize your interaction experience by enabling or disabling haptic feedback. This new toggle can be found in the main "Settings" menu accessible from the side navigation bar.
![screenshot](/tedee-release-notes/docs/android/assets/1.239.0_haptic_feedback.png)

## Remote door sensor status monitoring is now available for administrators via the Tedee Bridge.
Administrators can now check the status of their Tedee Door Sensor even when away from home. When the door sensor is paired with a Tedee Lock and connected via a Tedee Bridge, the app will automatically display the sensor state over a remote connection if Bluetooth is unavailable.

## Auto-unlock: disable device movement check
A new option in Auto-unlock advanced settings allows users to disable the device movement check on Android, giving more control over auto-unlock behavior.
![screenshot](/tedee-release-notes/docs/android/assets/1.239.0_movement_settings.jpg)

## Added a visual warning and updated instructions for fingerprint enrollment on Tedee Keypad Pro.
To streamline the fingerprint enrollment process for the Tedee Keypad Pro, we’ve added a new visual warning that appears if a user taps their phone screen when they should be placing their finger on the keypad's scanner. We have also updated the on-screen descriptions to provide clearer guidance throughout the setup.
![screenshot](/tedee-release-notes/docs/android/assets/1.239.0_scan_fingerprint.jpg)

## Fixes
- Battery level now updates correctly without needing to close and reopen settings.
- Fixed an issue where the app displayed an old firmware version and re-attempted the same update after a successful upgrade.
